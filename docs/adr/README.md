# Architecture Decision Records

ADRs dokumentieren tragende Entscheidungen, ihren Kontext und ihre Folgen. Akzeptierte Entscheidungen werden nicht still überschrieben. Eine Änderung erhält ein neues ADR, das die vorherige Entscheidung ersetzt.

## Statuswerte

- Proposed
- Accepted
- Superseded
- Rejected
